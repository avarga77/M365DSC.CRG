SHORT DESCRIPTION
    Microsoft365Dsc Composite Resource Generator

LONG DESCRIPTION
    This module will be able to generate Composite Resources for Microsoft365DSC, which can be used on Microsoft365DSC deployments.
    These Composite Resources are used in the Microsoft365DSC whitepaper "Managing Microsoft 365 in true DevOps style with Microsoft365DSC and Azure DevOps" (https://aka.ms/m365dscwhitepaper).

EXAMPLES
    PS C:\> New-CompositeResourceModule -OutputPath C:\Temp
